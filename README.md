# RIPv2 

### Protocolo de enrutamiento RIPv2
 
* Vector distancia classles y soporta VLSM

* Incluye la proxima dirección de salto en actualizaciones

* Actualizaciones por multicast

* Autenticación opcional

* Máximo de saltos 15
